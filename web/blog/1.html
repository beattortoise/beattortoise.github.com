了解prototype

var A = {  name: 'a'}

A.propotype = {        //原型方法    A.method2 = function(){}  // 类方法      //原型方法实例后可用  

method: function(){ return this.name + '我是A'},  

method2: function(){}

}

调用： var a= new A();

a.mechod();    //a我是A

 

var B = {  name: 'b'}

A.propotype = {        

method: functino(){ return this.name + '我是B'}

}

B.prototype = new A();     

var b = new B();

b.method()  //b我是B

B的prototype为A的一个实例，可以理解B将A中的方法和属性全部克隆了一遍。可以出现这种情况：B的prototype是A的实例，同时A的prototype也是B的实例。

PS: B本体有method方法就会运行本体的方法，A的method是不会被克隆的。

var a = new A();  //我B也有这样的方法，但是我想用A的(B不用自己用A的方法)

a.method.call(b);  //通过call劫持this的对象 //b我是A
